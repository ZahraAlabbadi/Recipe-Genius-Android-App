## Overview
Recipe Genius is an Android application that generates personalized recipes using the OpenAI API based on user preferences such as ingredients, calories, and number of people.

## Features
- Chat-based recipe generator
- Ingredient-based search
- User authentication
- Chat history storage
- Offline storage using SQLite

## Technologies
Android Studio, Java, OpenAI API, SQLite

## Architecture
Modular structure separating UI, API calls, and local database logic.

## Challenges

- Managing chat history efficiently
- Improving user experience
